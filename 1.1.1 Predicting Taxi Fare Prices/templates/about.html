About This Project

This project predicts NYC taxi fares using machine learning. A RandomForestRegressor model was trained on the NYC Taxi dataset.

Methodology

  • Feature engineering: Extracted datetime features (hour, day, month).
  • Calculated distance using Haversine formula.
  • Filtered outliers and cleaned the dataset.
  • Trained RandomForestRegressor on processed data.
