About This Project

Project Overview

The House Price Predictor is designed to estimate housing prices based on critical features such as square footage, location, number of bedrooms, and amenities. By leveraging machine learning models, the tool helps individuals, developers, and investors make data-driven real estate decisions.

Objectives

  • Provide accurate and reliable housing price predictions.
  • Support students, researchers, and professionals with accessible ML tools.
  • Encourage transparency by making datasets and models openly available.

Methodology

The predictor is powered by a regression-based machine learning pipeline. The dataset was preprocessed to handle missing values, normalize numerical features, and encode categorical variables. After experimenting with multiple models, the final pipeline was optimized and saved using joblib for deployment.

Technologies Used

  • Python – data processing & model building
  • Pandas, NumPy – feature engineering
  • Scikit-learn – machine learning pipeline
  • Flask – backend framework for deployment
  • HTML, CSS, JS – frontend design
  • Kaggle dataset – training data
