Dataset Overview

This project is powered by the EuroSAT RGB dataset — a benchmark dataset for land use and land cover classification derived from Sentinel-2 satellite imagery.

About the Dataset

The EuroSAT RGB dataset contains 27,000+ labeled satellite images across 10 land use categories — including Residential, Industrial, River, Forest, and Highway. Each image represents a 64×64 RGB tile sourced from Sentinel-2 data, capturing rich spatial and spectral patterns.

10 Classes

Urban, Agricultural, Forest, River, Industrial, and more.

64×64 RGB

Standardized for lightweight CNN training.

27,000+ Images

Curated and balanced for diverse land use coverage.

Preprocessing Notes
  • Images were resized to 64×64 pixels.
  • Converted to RGB channels for uniformity.
  • Pixel values normalized to [0, 1] for stable CNN convergence.
