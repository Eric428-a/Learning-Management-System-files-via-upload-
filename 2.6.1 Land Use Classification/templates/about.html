Land Use Lab

Lightweight Land Use Classification using Keras CNN

About this Project

This project demonstrates a compact yet powerful Convolutional Neural Network (CNN) for land use classification. Built with Keras, it uses only three convolutional blocks while maintaining impressive accuracy and efficiency. The goal: to make geospatial AI models lightweight enough to run anywhere — even on limited hardware.

Model Metadata

  • Model file: models/eurosat_cnn_model.joblib
  • Model type: keras_object_joblib
  • Input shape: (64, 64, 3)
  • Output classes: 10
  • File size: 7.88 MB
  • Architecture:
    3 × (Conv2D → MaxPool)
    Flatten → Dense(128, ReLU) → Dropout(0.5) → Dense(10, Softmax)
  • Total parameters: ~2M
